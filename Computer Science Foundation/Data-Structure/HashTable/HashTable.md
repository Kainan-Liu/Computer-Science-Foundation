# HashTable

哈希表[散列表]，是一种基于数组和链表实现的数据结构，它通过建立**key-value之间的映射**，实现元素的**高效查询**。能够在O(1)时间内获取相对应的值

| 操作         | 数组 | 链表 | 哈希表 |
| ------------ | ---- | ---- | ------ |
| 尾部添加元素 | O(1) | O(1) | O(1)   |
| 查询元素     | O(n) | O(n) | O(1)   |
| 删除元素     | O(n) | O(n) | O(1)   |

**哈希表进行增删改查的时间复杂度都是O(1)**
